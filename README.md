### Project Description
CineWatch is a movie discovery website that allows users to browse trending and currently playing movies,search for titles, and save movies they are interested in to their own personal space.

The application has a modern and user friendly interface that lets users:

-Browse trending movies that are updated regularly

-Explore movies currently playing in theaters around the world

-Save favorite movies to a personal collection called My Space

-View curated movie recommendations and reviews

-Search and explore movie details fetched dynamically from an external API

#### Pages Included

-**Home Page**

-**Browse Page**

-**My Space**

A consistent navbar is available across all pages for easy navigation.

This project was built as part of a frontend web development assignment, with a focus on responsive design, API integration, modern JavaScript , and creating a clean and interactive user experience.

## Custom Frontend Requirement
### Responsive card grid with Hover effects 
This was implemented using movie cards that automatically adjust based on screen size while maintaining a clean layout across desktop, tablet, and mobile devices.

On the home and browsing pages, hovering over a card slightly zooms in the movie poster and reveals additional details such as the IMDb rating and action buttons (like viewing details or saving the movie to My Space).

In the My Space page, the card layout is slightly different and focused more on quick management. Instead of hover-based details, it uses a set of action icons.
